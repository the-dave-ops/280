## 🔧 שינויים טכניים

### Grid Layout:
```typescript
// Before:
<div className="grid grid-cols-6 gap-1.5">
  {/* 6 number fields */}
</div>
<div className="grid grid-cols-3 gap-1.5 mt-1.5">
  {/* 3 select fields */}
</div>

// After:
<div className="grid grid-cols-9 gap-1">
  {/* 6 number fields + 3 select fields */}
</div>
```

### Spacing:
- **Padding**: `p-2` → `p-1.5` (חיסכון במקום)
- **Gap**: `gap-1.5` → `gap-1` (צפיפות גבוהה יותר)
- **הסרת**: `mt-1.5` (אין צורך בשורה שנייה)
